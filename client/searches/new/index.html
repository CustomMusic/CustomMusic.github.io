How To Search and License

  1. Search for tracks using dropdowns or keyword box.
  2. Add tracks to "Folder" icon in "Action" column. Create a new "Quick Project" per production.
  3. Click "Quick Projects" to view, edit and email.
  4. Click "Checkout" and select "Quick Project" to license.
